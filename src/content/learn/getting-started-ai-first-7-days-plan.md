If you are new to AI, do not start with model internals.
Start with useful habits.

## Day 1: Learn the core vocabulary

Understand five terms:

- model
- prompt
- context
- hallucination
- retrieval

You only need enough to ask better questions.

## Day 2: Use one assistant for one real task

Pick a concrete task (email draft, summary, study notes) and compare:

- your first attempt
- AI-assisted attempt

Measure time and quality difference.

## Day 3: Practice prompt iteration

Try the same task with three prompt styles:

- direct request
- role + constraints
- examples + desired format

You will feel why specificity matters.

## Day 4: Verify outputs

Build the reflex: never trust first output blindly.

Check:

- facts
- missing assumptions
- tone/fit for audience

## Day 5: Build a repeatable template

Create one reusable prompt for your most frequent task.

## Day 6: Add one automation

Use AI inside tools you already use (docs, email, coding, notes).

## Day 7: Review and codify

Write your personal “AI playbook”:

- what tasks AI helps with
- where it fails
- your verification checklist

## The real goal

Beginner success is not “knowing everything about AI.”
It is consistently using AI to improve outcomes while keeping human judgment in charge.
